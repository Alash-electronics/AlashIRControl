# Примеры на русском языке

Эта папка содержит примеры использования библиотеки AlashIRControl с комментариями на русском языке.

## Доступные примеры

### Приёмник (Receiver)
- **receiverKeyPress** - Получение кода при нажатии кнопки на пульте
- **receiverKeyHold** - Получение кода при удержании кнопки на пульте
- **receiverProtocol-Info** - Вывод информации о протоколе принятого сигнала
- **receiverProtocol-InfoAll** - Подробный вывод всей информации о протоколе
- **receiverSetProtokol** - Установка пользовательского протокола приёма

### Передатчик (Transmitter)
- **transmitterWithoutRetries** - Отправка IR-кода без повторов
- **transmitterWithRetries** - Отправка IR-кода с повторами при удержании кнопки
- **transmitterSetProtokol** - Установка пользовательского протокола передачи

### Примеры с готовыми пультами
- **BlackRemoteControllerKeys** - Коды для чёрного пульта (17 кнопок)
- **GrayRemoteControllerKeys** - Коды для серого пульта (21 кнопка)

## Как использовать

1. Откройте нужный пример в Arduino IDE
2. Подключите ваш IR приёмник или передатчик к соответствующему пину
3. Загрузите скетч на Arduino
4. Откройте Serial Monitor для просмотра результатов
