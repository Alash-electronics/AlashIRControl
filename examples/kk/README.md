# Қазақ тіліндегі мысалдар

Бұл қалтада AlashIRControl кітапханасын пайдалану мысалдары қазақ тіліндегі түсініктемелермен берілген.

## Қолжетімді мысалдар

### Қабылдағыш (Receiver)
- **receiverKeyPress** - Пультте батырма басылғанда кодты алу
- **receiverKeyHold** - Пультте батырма басылып тұрғанда кодты алу
- **receiverProtocol-Info** - Қабылданған сигнал хаттамасы туралы ақпаратты шығару
- **receiverProtocol-InfoAll** - Хаттама туралы барлық ақпаратты толық шығару
- **receiverSetProtokol** - Қабылдау үшін пайдаланушы хаттамасын орнату

### Таратқыш (Transmitter)
- **transmitterWithoutRetries** - ИҚ-кодын қайталаусыз жіберу
- **transmitterWithRetries** - Батырма басылып тұрғанда ИҚ-кодын қайталаумен жіберу
- **transmitterSetProtokol** - Жіберу үшін пайдаланушы хаттамасын орнату

### Дайын пульттермен мысалдар
- **BlackRemoteControllerKeys** - Қара пультінің кодтары (17 батырма)
- **GrayRemoteControllerKeys** - Сұр пультінің кодтары (21 батырма)

## Қалай пайдалану керек

1. Arduino IDE-де қажетті мысалды ашыңыз
2. ИҚ-қабылдағышыңызды немесе таратқышыңызды сәйкес пинге қосыңыз
3. Скетчті Arduino-ға жүктеңіз
4. Нәтижелерді көру үшін Serial Monitor-ды ашыңыз
